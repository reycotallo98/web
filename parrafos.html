Bienvenidos a mi página web

Desde hace años todo texto en un documento HTML debe ir dentro de una etiqueta que sirva para decir qué tipo de texto es. Y eso significa, que el texto debe de ir dentro de una etiqueta de párrafo. Se comentan las etiquetas disponibles para marcar párrafos.

Formatear un texto pasa por tareas tan evidentes como definir los párrafos, justificarlos, introducir viñetas, numeraciones o bien poner en negrita, itálica...

En esta primera página, trabajararemos con los parrafos
Para realizar un salto de linea usamos la etiqueta br como hemos visto
Tambien podemos crear distintos encabezados como vais a ver coon sus respectivas etiquetas, estos se ajustan automaticamente de tamaño.

Para poner titulos usaremos estas etiquetas

H1

H2

H3

H4

H5
H6

Ademas podemos añadirles efectos como poner el texto en negrita con la etiqueta strong y para ponerla en cursiva usaremos la etiqueta em

Si queremos crear superindices x2 usaremos la etiqueta sup, y para los subindices, x2 usaremos la etiqueta sub

Podemos ademas crear abreviaturas que muestren su significaaco con la etiqueta abbr por ejemplo ONU
